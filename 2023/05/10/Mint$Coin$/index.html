Welcome to join us!

As a high-yield project that takes high-value user activity as its main profit point and shares all profits with community members after removing the necessary costs for community maintenance, we require that every user who participates in Mint meets our high-value user standards.

In addition, to prevent witch attacks, we also need to ensure that “high-value users” who participate in Mint are not false address identities forged by robots.

Here are our criteria for these requirements:

  1. When Minting, you must use Trust-Wallet and BSC network.
  2. When Minting, the wallet address must have more than 2 ETH assets.
  3. There cannot be any identical IP access between any two high-value user addresses.
  4. There cannot be any on-chain transaction records between any two high-value user addresses.

As long as the addresses that meet these conditions participate in the current round of Minting, each address will receive $XCoin$ worth 20U. Moreover, the more ETH an address has, the more $XCoin$ it will receive after this round of Minting. The logic is that XCoin = X * 10, where X is the amount of ETH you own.
